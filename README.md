# Python Mini Projects

A collection of beginner-friendly Python projects focused on building logic, improving problem-solving skills, and strengthening core programming concepts.

---

## Hangman Game

A classic command-line word-guessing game played in the terminal where the player tries to guess a hidden word one letter at a time before running out of attempts.

**Features:**
- Random word selection  
- Input validation  
- Tracks correct and incorrect guesses  
- Simple and interactive gameplay
